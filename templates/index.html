About

Assignment Tracker is a simple Flask web app created as a school project. It allows users to create an account, manage their assignments, and track due dates and progress. The application provides a straightforward interface for students to organize and monitor their tasks in one place. The primary goal of this project is to provide a simple, intuitive tool for students to track their assignments. The project is built as a learning exercise in web development with Flask, with a focus on database handling. The website is open source and avaliable on github.
